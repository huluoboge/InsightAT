# Parallel Hybrid SfM 架构设计总结

---

# 1. 设计目标

本系统面向：

- 航空影像
- 倾斜摄影
- 大规模数据（上万 / 数万张影像）

核心目标：

1. **拓扑优先，而非第一层极致精度**
2. **强并行能力**
3. **高效率、可扩展**
4. 为后续 Global 优化提供稳定初值

---

# 2. 核心设计哲学

## 2.1 分层思想

系统分为两层：

### 第一层：粗 SfM 结果（Cluster + Merge + Global BA）

- **目标**：建立一份**粗的、全局一致的 SfM 结果**，为第二层提供稳定初值。
- **流程**：**Cluster 内并行重建 → Merge（融合/对齐）→ Global BA**。
  - Cluster 划分后，各 Cluster 内独立做增量式重建（局部一致）。
  - Merge 阶段：基于共视对齐各 Cluster（如 Sim3 / Pose Graph），统一坐标系与尺度。
  - **本层即使用 Global BA**：在 merge 之后对全体位姿与三维点做一次 Global BA，得到粗的全局一致解。
- **特点**：可降分辨率、控制特征数量以控制规模；拓扑正确、全局一致，但尚未达到最终精度。
- **输出**：粗位姿、粗三维点，供第二层做位姿引导匹配与高精度优化。

### 第二层：高精度 SfM（规划中）

- **目标**：在第一层粗结果基础上，达到**高精度**的位姿与结构。
- **计划内容**（待具体实现）：
  - **原始分辨率特征**：不再降采样，使用全分辨率图像上的特征。
  - **位姿引导的匹配**：利用第一层位姿缩小匹配搜索范围、提高可靠性。
  - **高精度相对位姿**：子像素/最小二乘匹配、精细几何估计。
  - **Global BA 策略**：在高精度观测与约束下再次进行全局 BA（或相应全局优化策略）。
- **与第一层关系**：第一层提供初值与匹配引导；第二层在此基础上做精度提升。

---

## 2.2 利用航空数据的统计特性

航空 / 倾斜摄影具有：

- 高重叠率
- 规则航线
- 大量高 vis 三维点
- 强连通图结构

因此：

> 强 seed 是天然存在的，应主动利用。

采用 Hybrid 策略：

- 统计稳定性选 seed
- 增量式扩张
- 局部 BA 稳定结构

---

# 3. 系统总体流程

---

## Stage 0：构建邻接图

使用 F / E / H 估计结果构建 view graph。

图 G = (V, E)

V: 图像
E: 几何验证通过的图像对

---

## Stage 1：图划分（并行核心）

目标：

- Cluster 并集 = 全集
- Cluster 交集 = ∅
- 每个 Cluster 规模 ≈ 500–1000 张
- 内部连通强
- 外部边尽量少但保留桥接

每个 Cluster 额外加入：

### Neighbor 图像

- 与 Cluster 内强连接但属于外部
- 不属于主集合
- 作为缓冲与融合桥梁

作用：

1. 防止 Cluster 重建失败
2. 提供跨 Cluster 约束
3. 提升局部鲁棒性

---

## Stage 2：Cluster 内并行增量重建

每个 Cluster 独立执行：

- Seed 选择（基于 vis / inlier / baseline）
- 增量式扩张
- Local BA
- 允许个别图像失败

输出：

- 局部一致的位姿
- 局部三维点

特性：

- 坐标系彼此独立
- 存在尺度 / 漂移差异

---

## Stage 3：Cluster 融合（Merge）

目标：统一所有 Cluster 坐标系与尺度，得到单一全局图。

步骤：

### Step 1：Cluster 级对齐
- 基于共视图像
- 估计 Sim3
- 粗对齐，统一尺度与坐标系

### Step 2：Image-level Pose Graph 优化（可选中间步）

变量：所有图像位姿  
边：共视约束  
优化：SE3 或 Sim3，不包含三维点  
目标：平均漂移、统一尺度，为后续 Global BA 提供更好初值。

---

## Stage 4：第一层 Global BA 与结构修复

- **Global BA**：在 merge 后对**全体位姿 + 三维点**做一次全局光束法平差，得到第一层的**粗 SfM 结果**（全局一致、拓扑正确）。
- 结构修复：重新三角化、处理失败图像（PnP + Local BA）等。
- 第一层输出：粗位姿、粗三维点，供第二层使用。

---

# 4. 关键策略

## 4.1 第一层：粗而全局一致

第一层目标不是最终精度，而是：

> 拓扑正确 + 全局一致（经 Merge + Global BA）

通过 Cluster 内并行 + Merge + **第一层 Global BA**，得到一份可用的粗 SfM 结果，供第二层做位姿引导与高精度优化。

---

## 4.2 失败图像后置处理

策略：

- Cluster 内允许个别图像失败
- 融合 / Global BA 前后再补
- 使用 PnP + Local BA 修复

避免早期阻塞系统。

---

## 4.3 第一层即使用 Global BA

- 第一层在 **Cluster + Merge** 之后，对全体做 **Global BA**，得到粗的全局一致解。
- 这样第一层输出已是全局一致的位姿与点，便于第二层做位姿引导匹配与高精度优化。
- 第二层将再次采用高精度观测与 Global BA（或相应全局策略）做最终精化。

---

# 5. 系统优势

- 第一层：高并行（Cluster 内）、可扩展、适合超大规模数据；Merge + Global BA 得到粗的全局一致结果。
- 第二层：在第一层初值上做原始分辨率、位姿引导匹配与高精度 Global BA，兼顾效率与最终精度。

---

# 6. 核心思想总结

**第一层（粗 SfM）**

> Cluster 内并行重建 → Merge 统一坐标系与尺度 → **Global BA** → 得到粗的全局一致位姿与点。

**第二层（高精度 SfM，规划中）**

> 原始分辨率特征 + 位姿引导匹配 + 高精度相对位姿 + **Global BA 策略** → 最终高精度结果。

整体是：

- **第一层**：Cluster + Merge + Global BA，产出粗 SfM。
- **第二层**：在第一层基础上，原始分辨率、位姿引导、高精度观测与全局优化。
