# 2度点（2-View Track）在Global BA中的差异化策略

## 背景与问题

在增量SfM的全局BA阶段，track_store中存在大量仅被两张图像观测的2度点（track length = 2）。这类点在BA中的特性：

- **高成本**：每个2度点引入3个3D变量 + 4个residual（每个观测2维）
- **低信息**：仅提供1条epipolar约束，对全局结构的贡献接近零（当两张图像已有足够多3度+点时）
- **稳定性隐患**：三角化精度取决于基线角度，小基线的2度点XYZ极不稳定

根据 `diagnose_track_structure` 的日志和ChatGPT分析，2度点容易占总track数的60-80%，是BA观测量膨胀的主要来源。

**核心思路**：不删除2度点，而是在全局BA构建BAInput时，对"可以跳过"的2度点打标，不加入BA，但保留用于BA后的快速重新三角化。

---

## 策略设计

### 判定依据：Track稳定性评分 S(track)

对每个2度点track，以当前已有的位姿和3D坐标计算几何质量：

```
S(track) = sin²(θ)
```

其中 θ 是该track唯一两个观测射线之间的三角化角度（parallax angle），从当前 `poses_C` 和 track 的3D坐标直接计算：

```cpp
ray0 = (X - C0).normalized()
ray1 = (X - C1).normalized()
cos_θ = ray0.dot(ray1)
θ = acos(cos_θ)
S = sin²(θ)
```

S越高 → 基线越大 → 点的深度精度越高 → 对BA的贡献越冗余（两张图之间已有足够约束）。

### 图像稳定性判断 image_stable(i)

投影track稳定性到图像：

```
obs_count(i) = 图像i上有效三角化观测总数
stable_obs_count(i) = 图像i上 S(track) > S_threshold 的观测数
ratio_stable(i) = stable_obs_count(i) / obs_count(i)
image_stable(i) = (obs_count(i) >= min_stable_obs) && (ratio_stable(i) >= stable_ratio_threshold)
```

### 2度点跳过条件

```
skip_from_global_ba(track) =
    track.length == 2 （在当前registered图像中只有2个有效观测）
    AND S(track) > S_threshold           // 自身几何质量好
    AND image_stable(im_A)               // 两张观测图像都已稳定
    AND image_stable(im_B)
```

满足条件的2度点：**不加入BAInput的points3d和observations**，但保留在TrackStore中（track不删除，flag不变）。

### BA后的快速重新三角化

Global BA完成后，对所有被skip的2度点执行轻量三角化：直接用更新后的位姿和两条射线DLT三角化，检查重投影误差，通过则更新track的XYZ。这一步已有 `run_retriangulation` 基础设施，被skip的2度点会在主循环中已有的 `enable_full_scan_after_global_ba` 的kFullScan中被自动重三角化，**不需要额外步骤**。

### 适用范围限制

| BA类型 | 是否应用此策略 |
|--------|--------------|
| `run_global_ba` (via `build_ba_input_from_store`) | ✅ 应用 |
| `run_local_ba_colmap` (via `build_ba_input_colmap_local`) | ❌ 不应用 |
| `run_local_ba_batch_neighbor` | ❌ 不应用 |
| `run_local_ba` (kWindow) | ❌ 不应用 |

**理由**：local BA的目的是稳定刚加入的新图像，此时2度点可能是该图像唯一约束；只有在global BA中，跳过高质量2度点才是安全的。

## 实现步骤

### Step 1：新增配置参数到 `GlobalBAOptions`

文件：`src/algorithm/modules/sfm/incremental_sfm_pipeline.h`

在 `GlobalBAOptions` 结构体中新增：

```cpp
// ── 2-degree track pruning in global BA ──────────────────────────────────────
/// Enable skipping 2-degree tracks from global BA when both observing images are stable.
bool skip_2degree_tracks = false;
/// Minimum sin²(parallax_angle) for a 2-degree track to be considered "skippable".
/// Tracks with S < this threshold are still included in BA (weak geometry, important constraint).
/// Typical: 0.001 ≈ sin²(1.8°); 0.002 ≈ sin²(2.6°); 0.003 ≈ sin²(3.1°)
double skip_2degree_min_angle_score = 0.002;
/// Minimum number of valid triangulated observations for an image to be considered "stable".
int skip_2degree_min_stable_obs = 50;
/// Minimum fraction of high-quality (S > threshold) observations for an image to be "stable".
double skip_2degree_stable_ratio = 0.5;
```

### Step 2：添加 `compute_image_stability` 辅助函数

文件：`src/algorithm/modules/sfm/incremental_sfm_pipeline.cpp`（anonymous namespace内）

```cpp
/// For each registered image, compute whether it has enough high-quality track support
/// to allow skipping 2-degree tracks in global BA.
/// S(track) = sin²(parallax_angle) computed from poses_C and track XYZ.
static std::vector<bool> compute_image_stability(
    const TrackStore& store, const std::vector<Eigen::Vector3d>& poses_C,
    const std::vector<bool>& registered,
    double min_angle_score, int min_stable_obs, double stable_ratio) {
  const int n = store.num_images();
  std::vector<int> obs_count(n, 0);
  std::vector<int> stable_count(n, 0);
  std::vector<Observation> obs_buf;
  for (size_t ti = 0; ti < store.num_tracks(); ++ti) {
    const int tid = static_cast<int>(ti);
    if (!store.is_track_valid(tid) || !store.track_has_triangulated_xyz(tid)) continue;
    obs_buf.clear();
    store.get_track_observations(tid, &obs_buf);
    // Count only registered observations
    std::vector<int> reg_obs_imgs;
    for (const auto& o : obs_buf) {
      const int im = static_cast<int>(o.image_index);
      if (im >= 0 && im < n && registered[im]) reg_obs_imgs.push_back(im);
    }
    // Compute S only for 2-degree tracks (exactly 2 registered observations)
    double S = 1.0; // default high for higher-degree tracks
    if (reg_obs_imgs.size() == 2) {
      float tx, ty, tz;
      store.get_track_xyz(tid, &tx, &ty, &tz);
      const Eigen::Vector3d X(tx, ty, tz);
      const Eigen::Vector3d r0 = (X - poses_C[reg_obs_imgs[0]]).normalized();
      const Eigen::Vector3d r1 = (X - poses_C[reg_obs_imgs[1]]).normalized();
      const double cos_a = std::max(-1.0, std::min(1.0, r0.dot(r1)));
      const double sin_a = std::sqrt(1.0 - cos_a * cos_a);
      S = sin_a * sin_a;
    }
    for (int im : reg_obs_imgs) {
      obs_count[im]++;
      if (S >= min_angle_score) stable_count[im]++;
    }
  }
  std::vector<bool> stable(n, false);
  for (int i = 0; i < n; ++i) {
    if (!registered[i]) continue;
    stable[i] = (obs_count[i] >= min_stable_obs) &&
                (obs_count[i] > 0) &&
                (static_cast<double>(stable_count[i]) / obs_count[i] >= stable_ratio);
  }
  return stable;
}
```

### Step 3：修改 `build_ba_input_from_store` 签名，接受可选过滤参数

在 `build_ba_input_from_store` 函数签名中增加两个可选参数：

```cpp
bool build_ba_input_from_store(
    ...,       // 现有参数不变
    const std::vector<bool>* image_stable_for_2degree_skip,  // nullptr = no skip
    double skip_2degree_min_angle_score,
    int* n_skipped_2degree_out);  // 可选：输出skip数量
```

在track迭代循环中，`visible_in_ba` 计算之后，在 `if (visible_in_ba < 2) continue;` 之后插入：

```cpp
// 2-degree track pruning: skip if both observing images are stable and geometry is strong.
if (image_stable_for_2degree_skip && visible_in_ba == 2) {
  // Find the two registered observers
  int im_a = -1, im_b = -1;
  for (const auto& o : obs_buf) {
    int im = static_cast<int>(o.image_index);
    if (ba_global_set.count(im)) {
      if (im_a < 0) im_a = im; else im_b = im;
    }
  }
  if (im_a >= 0 && im_b >= 0 &&
      static_cast<size_t>(im_a) < image_stable_for_2degree_skip->size() &&
      static_cast<size_t>(im_b) < image_stable_for_2degree_skip->size() &&
      (*image_stable_for_2degree_skip)[im_a] && (*image_stable_for_2degree_skip)[im_b]) {
    // Compute S = sin²(θ) from current XYZ and camera centres
    float tx, ty, tz;
    store.get_track_xyz(track_id, &tx, &ty, &tz);
    const Eigen::Vector3d X(tx, ty, tz);
    const Eigen::Vector3d r0 = (X - poses_C[im_a]).normalized();
    const Eigen::Vector3d r1 = (X - poses_C[im_b]).normalized();
    const double cos_a = std::max(-1.0, std::min(1.0, r0.dot(r1)));
    const double sin_a = std::sqrt(1.0 - cos_a * cos_a);
    const double S = sin_a * sin_a;
    if (S >= skip_2degree_min_angle_score) {
      if (n_skipped_2degree_out) ++(*n_skipped_2degree_out);
      continue;  // 不加入BAInput
    }
  }
}
```

### Step 4：在 `run_global_ba` 中调用

在 `run_global_ba` 函数中，调用 `build_ba_input_from_store` 前：

```cpp
// Precompute image stability for optional 2-degree track pruning
std::vector<bool> image_stable_2deg;
int n_skipped_2deg = 0;
const bool do_skip_2deg = (global_ba_2deg_opts != nullptr && global_ba_2deg_opts->skip_2degree_tracks);
if (do_skip_2deg) {
  image_stable_2deg = compute_image_stability(*store, *poses_C, registered,
      global_ba_2deg_opts->skip_2degree_min_angle_score,
      global_ba_2deg_opts->skip_2degree_min_stable_obs,
      global_ba_2deg_opts->skip_2degree_stable_ratio);
}
```

## 关键风险与缓解

| 风险 | 缓解措施 |
|------|---------|
| 弱连接图像被剥夺BA约束 | `image_stable` 检查确保两端图像都已有足够高质量观测；新注册图像obs_count < min_stable_obs，自动保留 |
| 小基线2度点被错误跳过 | S = sin²(θ) 对小基线极敏感，小角度时S接近0，远低于threshold，强制保留 |
| 初始阶段过激跳过 | `min_stable_obs=50`，早期图像obs少，自然不触发 |
| local BA中误用 | 只修改 `build_ba_input_from_store`（全局BA路径），local BA路径不受影响 |
| kFullScan后S失效 | skip列表不持久化，下次global BA重新评估，自动适应位姿变化 |

---

## 预期效果

以500张图像的航测场景为例，若2度点占总点数60%，其中50%满足skip条件：

- BA的 `n_points` 减少约30%
- `n_observations` 减少约20-30%
- BA时间（Schur complement主导）预计下降30-50%
- 精度理论上不变（skip的都是已有充足约束的几何）

---

## 配置建议

```
global_ba.skip_2degree_tracks = false          # 默认关闭，先验证
global_ba.skip_2degree_min_angle_score = 0.002 # sin²(约2.6°)
global_ba.skip_2degree_min_stable_obs = 50     # 图像至少有50个高质量观测
global_ba.skip_2degree_stable_ratio = 0.5      # 其中50%是高质量点
```

建议先在500张图像数据集上以 `--skip-2degree-tracks` 对比测试，确认RMSE无退化后设为默认开启。
