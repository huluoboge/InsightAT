# Parallel Hybrid SfM 架构设计总结

---

# 1. 设计目标

本系统面向：

- 航空影像
- 倾斜摄影
- 大规模数据（上万 / 数万张影像）

核心目标：

1. **拓扑优先，而非第一层极致精度**
2. **强并行能力**
3. **高效率、可扩展**
4. 为后续 Global 优化提供稳定初值

---

# 2. 核心设计哲学

## 2.1 分层思想

系统分为两层：

### 第一层：并行 Hybrid 增量重建
- 允许一定漂移
- 不追求全局最优
- 保证拓扑结构正确
- 输出稳定的位姿初值

### 第二层：全局融合与优化
- Pose Graph 融合
- 可选 Global BA
- 误差平均与尺度统一

---

## 2.2 利用航空数据的统计特性

航空 / 倾斜摄影具有：

- 高重叠率
- 规则航线
- 大量高 vis 三维点
- 强连通图结构

因此：

> 强 seed 是天然存在的，应主动利用。

采用 Hybrid 策略：

- 统计稳定性选 seed
- 增量式扩张
- 局部 BA 稳定结构

---

# 3. 系统总体流程

---

## Stage 0：构建邻接图

使用 F / E / H 估计结果构建 view graph。

图 G = (V, E)

V: 图像
E: 几何验证通过的图像对

---

## Stage 1：图划分（并行核心）

目标：

- Cluster 并集 = 全集
- Cluster 交集 = ∅
- 每个 Cluster 规模 ≈ 500–1000 张
- 内部连通强
- 外部边尽量少但保留桥接

每个 Cluster 额外加入：

### Neighbor 图像

- 与 Cluster 内强连接但属于外部
- 不属于主集合
- 作为缓冲与融合桥梁

作用：

1. 防止 Cluster 重建失败
2. 提供跨 Cluster 约束
3. 提升局部鲁棒性

---

## Stage 2：Cluster 内并行增量重建

每个 Cluster 独立执行：

- Seed 选择（基于 vis / inlier / baseline）
- 增量式扩张
- Local BA
- 允许个别图像失败

输出：

- 局部一致的位姿
- 局部三维点

特性：

- 坐标系彼此独立
- 存在尺度 / 漂移差异

---

## Stage 3：Cluster 融合（Pose Graph 层）

目标：统一所有 Cluster 坐标系。

步骤：

### Step 1：Cluster 级对齐
- 基于共视图像
- 估计 Sim3
- 粗对齐

### Step 2：Image-level Pose Graph 优化

变量：
- 所有图像位姿

边：
- 共视约束

优化：
- SE3 或 Sim3
- 不包含三维点

目标：
- 平均漂移
- 统一尺度

---

## Stage 4：结构重建与修复

1. 固定位姿
2. 重新三角化三维点
3. 处理失败图像（PnP + Local BA）
4. 可选小规模 BA（点 + 内参）

---

# 4. 关键策略

## 4.1 允许第一层漂移

第一层目标不是高精度，而是：

> 保证拓扑正确 + 误差可控

误差在 Pose Graph 层统一平均。

---

## 4.2 失败图像后置处理

策略：

- 第一阶段允许失败
- 融合后再补
- 使用 PnP + Local BA 修复

避免早期阻塞系统。

---

## 4.3 避免过早 Global BA

原因：

- 计算量大
- 破坏并行性
- 不符合效率优先目标

Global BA 仅作为最终 refinement（可选）。

---

# 5. 系统优势

- 高并行
- 可扩展
- 适合超大规模数据
- 充分利用航空数据统计特性
- 第一层计算成本可控

---

# 6. 核心思想总结

> 不追求一步到位的最优解。
>
> 先构建稳定局部结构。
>
> 再通过 Pose Graph 统一全局。
>
> 最后在需要时进行全局精化。

这是一个：

- 拓扑优先
- 分层优化
- 并行驱动
- 工程效率导向

的 SfM 系统架构。
